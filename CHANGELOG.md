# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0

- Initial Marketplace release.
- Save, restore, rename, delete, import, export, and compare context snapshots.
- Snapshot tree view and status bar integration.
- Quick-switch slots for recent snapshots.
- Optional automatic snapshot switching on Git branch changes.
- Improved restore flow, import refresh behavior, multi-root workspace handling, and branch-switch edge-case handling.
